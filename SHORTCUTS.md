# Whiteboard Shortcuts

This file lists the current keyboard controls available in the whiteboard.

## Drawing Mode (Ctrl Hold)

- `Hold Ctrl` (Windows/Linux) or `Hold Command` (Mac) + move pointer: draw (same as left-drag pen flow).
- `Release Ctrl/Command`: current stroke ends immediately.

## Tool Shortcuts

- `Ctrl + Q`: Pen
- `Ctrl + W`: Pen
- `Ctrl + E`: Eraser
- `Ctrl + Shift + Q`: Lasso Select
- `Ctrl + A`: Circle/Ellipse Shape
- `Ctrl + T`: Text Tool
- `Ctrl + R`: Laser Pointer

## Important Rule

- Any other `Ctrl/Command + key` combination is blocked inside the app (no whiteboard action).

## Mouse / Touchpad

- Left click / drag: normal writing and tool interaction.
- Two-finger pinch on supported devices: zoom.
- Two-finger scroll: pan board.
